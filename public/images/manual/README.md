# Imágenes del Manual del Usuario

Coloca aquí las capturas de pantalla para el manual. El sistema muestra un placeholder estilizado si la imagen no existe.

**Formato recomendado:** PNG, 1200px de ancho mínimo, optimizado para web.

## Lista de imágenes requeridas

| Archivo | Sección | Descripción |
|---------|---------|-------------|
| `inicio-plataforma.png` | Introducción | Vista general de la plataforma (landing o home) |
| `buscador-vista-general.png` | Buscador | Panel completo del buscador con filtros y tabla |
| `buscador-filtros.png` | Buscador | Detalle de los filtros: keyword, entidad, objeto, estado, departamento |
| `buscador-detalle-proceso.png` | Buscador | Modal/vista de detalle de un proceso con botones de acción |
| `dashboard-graficos.png` | Dashboard | Dashboard con KPIs y gráficos estadísticos |
| `dashboard-calendario.png` | Dashboard | Calendario mensual de seguimientos con colores de urgencia |
| `dashboard-mis-procesos.png` | Dashboard | Historial de procesos notificados con re-notificación |
| `premium-keywords.png` | Premium | Configuración de palabras clave en el perfil |
| `premium-score.png` | Premium | Resultado del score de compatibilidad IA |
| `bot-telegram-notificacion.png` | Bots | Captura del bot de Telegram con notificación y botones |
| `bot-whatsapp-chat.png` | Bots | Captura del chat de WhatsApp con notificación |
